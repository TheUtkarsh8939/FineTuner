## Introduction
Finetuner is a tool designed to help developers fine tune applications that depend on weights or other variables to acheive the best possible performance.

## Running Demo of Finetuner
To run the demo of Finetuner, follow the instructions below:

1. run `./Finetuner.exe init`
2. Choose Demo mode
3. run `./Finetuner.exe run ./config.json`
4. The application will start running and output the final result once complete
